Simple Page to Sidebar lets you simply add page content to a sidebar. No more, no less.

== Description ==

Simple Page to Sidebar is a plugin that adds a widget which enables you to display any page content in your sidebars. Just choose any page using the dropdown in the widget area. Also unpublished or private pages content can be shown.

= Features =

* No posts, *just pages*.
* Internationalized for *english and german*.

== Installation ==

1. Download this plugin and unzip it.
2. Upload the whole `simple-page-to-sidebar` folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to your 'Widget' section and simply add some pages to your sidebars.
